widgetframe: A package for wrapping htmlwidgets in responsive iframes.

Description

This pacakge provides two functions frameableWidget, and frameWidget. The frameableWidget is used to add extra code to a htmlwidget which allows is to be rendered inside a responsive iframe. The frameWidget is a htmlwidget which displays content of another htmlwidget inside a responsive iframe.
