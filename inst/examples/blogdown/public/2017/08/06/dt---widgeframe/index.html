DT + widgeframe

2017/08/06

The two examples below show how to properly embed a DT widget inside a blog post using the widgetframe package.

DT

library(DT)
library(widgetframe)
## Loading required package: htmlwidgets
dt <-  datatable(
  head(iris, 20), 
  options = list(
     columnDefs = list(list(className = 'dt-center', targets = 5)),
     pageLength = 5, lengthMenu = c(5, 10, 15, 20)),
  fillContainer = T)

frameWidget(dt, height = 350, width = '95%')
